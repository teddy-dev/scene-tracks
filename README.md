# 🔊 Scene Tracks for FoundryVTT
A super basic module to allow scenes to have their own background tracks that will play locally to players viewing the scene rather than a globally activated playlist.

## 📔 Usage
- Edit any scene, configure the scene background track under the Miscellaneous tab. _(Ambience -> Basic on v13)_
- Load the scene _(leave and come back if already on it!)_
## 💾 Installing
While still testing, you can install via the manifest url:  
```
https://raw.githubusercontent.com/teddy-dev/scene-tracks/refs/heads/main/module.json
```
